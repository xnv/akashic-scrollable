akahsic-scrollable Documentation

この文書は akashic-scrollable の簡単なガイドです。

akashic-scrollable は、Akashic Engine でエンティティのクリッピングとスクロールを実現するライブラリです。

このライブラリが提供するクラス Scrollable に、任意のエンティティを追加することで、 エンティティは自動的にクリッピングされ、スクロールバーやスワイプ操作でスクロールできるようになります。
