Class NinePatchVerticalScrollbar

The ninepatch image vertical scrollbar entity. Takes two images as ninepatch images for the background and the bar itself.

9パッチ画像の縦スクロールバー。 二つの画像(背景用・バー用)をとり、9パッチで拡大して利用する。

local

Inherited from E.local

このエンティティがローカルであるか否か。 コンストラクタで真が指定された時、または属するシーンがローカルシーンまたはローカルティック補間シーンである時、この値は真である。

    • Optional camera: Camera

      対象のカメラ。指定されなかった場合undefined
