onChangeBarPositionRate

onChangeBarPositionRate: Trigger<number>
  • Defined in ScrollbarLike.ts:14

The trigger must be fire()ed when the scroll position changed. The argument must be a number in [0, 1].

スクロール位置が変化した場合、 fire() されねばならないトリガー。 その引数は [0, 1] の数値でなければならない。

setBarProperties

  • setBarProperties(posRate?: number | null, contentLength?: number | null, viewLength?: number | null): void
    • Defined in ScrollbarLike.ts:26

    A method called by Scrollable when the properties are changed. The appeareance of the scrollbar must reflect the given properties.

    スクロールバーのプロパティが変化した時、 Scrollable から呼び出されるメソッド。 スクロールバーの表示内容は与えられたプロパティを反映せねばならない。

    Parameters

    • Optional posRate: number | null

      the scroll position rate ([0, 1]). スクロールポジション([0, 1] の数値)

    • Optional contentLength: number | null

      the length of the whole scrolled content. スクロールされる内容の長さ。

    • Optional viewLength: number | null

      the length of the displayed area. 表示される範囲の長さ。
